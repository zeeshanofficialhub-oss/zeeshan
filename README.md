# Nova Studio Website

A fully functional, responsive marketing website built with plain HTML, CSS, and JavaScript.

## Features

- Responsive layout for desktop and mobile devices
- Mobile navigation menu toggle
- Services and featured projects sections
- Project category filter buttons
- Rotating testimonial component
- Client-side contact form validation
- Contact form persistence to `localStorage`

## Run locally

Because this is a static site, you can open `index.html` directly in your browser.

Optional: run a local server from this folder:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.
